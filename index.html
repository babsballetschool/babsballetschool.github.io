Project Moderne dans gevorderd

We hebben weer een volgend dansproject, dit keer Modern gevorderd, Sarah Döker gaat het project verzorgen. Dit project duurt 2 zaterdagen (11 & 25 maart) van 14:00-16:00 (dubbel dansuur!) voor gevorderde dansers en kost totaal € 50,- voor leerlingen, mensen van buitenaf zijn ook welkom, zij betalen € 7,- extra administratie kosten. Ben snel met je inschrijving! Bij voldoende animo gaat het project door. Inschrijven kan via het formulier dat in de studio ligt of via de mail: babs@babsballetschool.nl
